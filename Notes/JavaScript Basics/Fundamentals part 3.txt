
PART 3

1. What are functions useful for?
    Functions are useful for code that is intended to be reusable.

2. How do you invoke a function?
    Function is invoked i.e exampleFunction(parameter1, parameter2)

3. What are anonymous functions?
    Anonymous functions are nameless functions  that expect to receive another function as a parameter.

4. What is function scope?
    Variables and things defined inside a function are inside their own separate scope.

5. What are return values?
    Return values are returned when the function has completed.

6. What are arrow functions?
    Arrow functions are handy for simple actions, especially for one-liners.
